Cargar nuevo cuestionario

Opciones:

  • Subir archivo JSON con preguntas (estructura compatible con la app).
  • O pegar/editar preguntas en el área de texto con el siguiente formato:
    Pregunta: ¿Qué es una expresión en programación?
    A) Un conjunto de reglas sintácticas.
    *B) Un fragmento de código que produce un valor.
    C) Una estructura de control que permite iteraciones.
    D) Una declaración con un punto y coma al final.
    Justificación: Una expresión en programación es cualquier fragmento de código que produce un valor, como 2 + 2 o "hola" + " mundo".
    ---
    Pregunta: ¿Cuál de las siguientes palabras clave en JavaScript introduce una constante?
    A) var
    B) let
    *C) const
    D) static
    Justificación: La palabra clave const define una constante cuyo valor no puede ser reasignado después de ser declarado.
    ---

Coloca un * delante de la opción correcta.
Separa cada pregunta con --- en una línea nueva.
Incluye la justificación después de las opciones.
